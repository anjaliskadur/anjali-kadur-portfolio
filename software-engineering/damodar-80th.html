Damodar's 80th Birthday Site

A mobile-first invitation site for Damodar Kadambi’s 80th birthday. The homepage opens on a full-bleed, alternating polaroid photo wall that shuffles on every visit, with event details and CTAs layered on top. Guests can RSVP online and contribute to a shared memory wall — photos, videos, or notes — without creating accounts. Edit/delete is handled with one-time browser tokens (plus an optional admin passcode for moderation). Built with Next.js, TypeScript, Tailwind, and Supabase so the family can collect RSVPs and keepsakes before and during the celebration. Live at damodar80th-neon.vercel.app.

Invitation Landing & Photo Wall

The landing experience is built around a full-bleed polaroid collage that reshuffles on each visit, with a centered invitation card carrying the date, venue, and primary actions — RSVP and Visit the Memory Wall. The stack is Next.js and TypeScript with Tailwind for a mobile-first layout, and Supabase for guest RSVPs and memory submissions secured by one-time browser tokens.

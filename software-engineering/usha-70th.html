Usha's 70th Birthday Site

Built a private, invitation-style website for a family birthday weekend in Houston. The site balances refined visual design with practical guest needs: dress code, hotel/venue info, and a Memory Wall where attendees can share messages, photos, and video notes if they can’t attend. Implemented with Next.js and TypeScript, styled in Tailwind, and connected to Supabase for real-time guest submissions with secure edit/delete for message owners. Focused on accessibility, mobile performance, and a calm, high-end experience for guests of all ages. Live at usha70th.vercel.app.

Invitation Landing & Guest Tools

The homepage leads with a calm, invitation-style hero framed by photo filmstrips, the event date and city, and clear paths to RSVP and the Memory Wall. Behind the visuals, guests get dress code and hotel/venue details, while Supabase powers real-time memory submissions with owner-only edit and delete — all tuned for accessibility and mobile performance across ages.
